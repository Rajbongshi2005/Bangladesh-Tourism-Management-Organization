![C++](https://img.shields.io/badge/Language-C%2B%2B-blue)
![Platform](https://img.shields.io/badge/Platform-Console-lightgrey)

A **console-based tourism management system** built in C++ to manage travel operations across Bangladesh. This project demonstrates **data structures, OOP, and algorithmic problem solving** through a real-world application.

## 🚀 Features

### 🔐 User Roles

* **Admin**

  * Manage tourists, assistants, and city data
  * Perform CRUD operations

* **Tourist**

  * View city info & tourist spots
  * Plan trips (shortest distance)
  * Book travel assistants

* **Travel Assistant**

  * Manage profile
  * View assigned यात्राएँ (trips)

---

### 🗺️ Smart Trip Planning

* Predefined **distance matrix for 64 districts**
* Minimum distance calculation
* Multi-city trip planning

---

### 🏨 Management System

* City database with tourist spots
* Booking & tracking system
* Travel history

---

## 🛠️ Tech Stack

## ⚙️ Installation & Usage

### 🔧 Requirements

* GCC / G++ Compiler

### ▶️ Run Locally

```bash
# Clone repository
git clone https://github.com/your-username/btmo.git

# Navigate to project
cd btmo

# Compile
g++ "Bangladesh Tourism Management Organization.cpp" -o tourism_system

# Run
./tourism_system
```
